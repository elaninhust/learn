## Java关键字

- 全部小写
- goto和const为保留字

## 标识符

>包、类、方法、变量和常量等的名称

- 包：作用类似于文件夹，区分不同文件夹下的名称相同的类
    + 全部小写
    + 用`.`分隔标识目录层级
- 类：
    + 每个单词的首字母均需大写
- 变量名：
    + 第一个单词的首字母小写，其余单词的首字母均需大写
- 常量：
    + 每个单词的字母均需大写
    + 单词和单词之间用下划线分隔`_`

## 进制

- 二进制：已`0b`或`0B`开头，如0b100
- 八进制：已`0`开头，如0100
- 十进制：默认是十进制，如100
- 十六进制：以`0x`或`0X`开头，如0x100

>各进制之间的相互装换：以十进制为桥梁，再进行转换。

1byte = 8bit
1k = 1024byte
1m = 1024k
1g = 1024m
1t = 1024g
